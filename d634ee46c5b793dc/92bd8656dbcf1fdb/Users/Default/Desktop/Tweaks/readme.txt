В этой папке буду помещать различные свои твики которые применяю как правило после установки системы,
или просто загрузившись в WinPE
Для изменения некоторых параметров реестра из под системы, необходимо запускать редактор с системными правами.
WinPE работает под системным аккаунтом, поэтому нет необходимости использовать различные программы для этого.
К тому же, если есть возможность это сделать из под WinPE, непосредственно после развертывания
системы, например программой WinNTSetup, то почему бы это не сделать сразу, не заморачиваясь потом.
Постеренно буду пополнять эту папку, по мере того как разгребать у себя..


Внимание:
По умолчанию диск на котором установлена Windows к которой применяется твики, указан С
Если в WinPE буква диска с Windows не С, измените букву на нужную.

строка 
set disk=C:\

где C:\ нужно изменить на нужную букву. 

А еще лучше, как делаю я, в случае необходимости просто меняю в WinPE букву нужного мне диска на С


Disable_WinDefender
Enable_WinDefender - отключение и включение Защитника Windows

Disable_UAC
Enable_UAC - отключение и включение контроля учетных записей (UAC) 

Disable_Prevent_Device_Encryption - отключение автоматического шифрования дисков BitLocker

sfc_scannow - проверка целостности офлайн Windows

Disable_Services - отключение некоторых ненужных служб
